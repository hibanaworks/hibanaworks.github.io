What if your compiler could prove your protocol is deadlock-free?

Hibana is a #![no_std] multiparty session types engine that brings established type-theoretic guarantees into everyday systems. Write your global choreography once, project it at compile time, and get affine cursors that make protocol violations impossible. 

// 1. Define the Global Choreography
const PING_PONG: g::Program<_> = g::seq(
    g::send::<Client, Server, Ping>(),
    g::send::<Server, Client, Pong>(),
);

// 2. Project to Role-Local Programs (Compile-Time)
const CLIENT: g::RoleProgram<'static, 0, _> = g::project::<0, _, _>(&PING_PONG);

// 3. Drive the Protocol (Runtime)
// The runtime ensures you MUST send Ping before you can recv Pong.
let (client, _) = client.flow::<Ping>()?.send(&100).await?;
let (client, pong) = client.recv::<Pong>().await?;

Const Projection

g::Program builds protocols with g::send / seq / par / route. Pure const fn composition without proc-macros.

Affine Cursors

Every step is a linear resource. You cannot skip a step, reuse a step, or drop a session mid-flight without explicit handling.

Transport Agnostic

Plug in TCP, UDP, QUIC, or in-memory channels via the BindingSlot trait.
