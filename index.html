What if your compiler could prove your protocol is deadlock-free?

Hibana is a #![no_std] multiparty session types engine that brings established type-theoretic guarantees into everyday systems. Write your global choreography once, project it at compile time, and get affine cursors that make protocol violations impossible. 

// 1. Define the Global Choreography
const PROTOCOL: g::Program<_> = g::seq(
    g::send::<Client, Server, Request>(),
    g::send::<Server, Client, Response>(),
);

// 2. Project to Role-Local Programs (Compile-Time)
const CLIENT: g::RoleProgram<'static, 0, _> = g::project::<0, _, _>(&PROTOCOL);
const SERVER: g::RoleProgram<'static, 1, _> = g::project::<1, _, _>(&PROTOCOL);

// 3. Drive the Protocol (Runtime)
// The compiler ensures you MUST send Request before you can recv Response.
let (client, _) = client.flow::<Request>()?.send(&payload).await?;
let (client, resp) = client.recv::<Response>().await?;

Const Projection

g::Program composes protocols with g::send, g::seq, g::par, g::route. Pure const fn composition without proc-macros.

Affine Cursors

Every step is a linear resource. You cannot skip a step, reuse a step, or drop a session mid-flight without explicit handling.

Deterministic Observability

20-byte lock-free tap ring records every control decision for deterministic replay and debugging.

Effect Policy Filter

An embedded bytecode VM that enforces dynamic policies (rate limits, access control) on every control transition.

Transport Agnostic

Plug in TCP, UDP, QUIC, or in-memory channels via the BindingSlot trait. Zero polling, drop-in compatibility.
